jQuery Pinit

a jQuery Pinterest plugin to automagically overlay 'Pin It!' buttons on your images

Demo

Usage

Just construct the jQuery object with the images that you want and invoke the .pinit() method. For example, to automagically assign a the 'Pin It' Button to the above images the following code was used:

		<ul id="demo">
			<li>
				<img src="http://mgiulio.altervista.org/wp-content/uploads/2011/03/pieces-of-me-300x225.jpg" alt="" title="pieces-of-me" width="300" height="225" class="aligncenter size-medium wp-image-416" />
			</li>
			<li>
				<img src="http://mgiulio.altervista.org/wp-content/uploads/2012/01/Playing-with-Open-Data-a-simple-viewer-for-DecoroUrbano.org_-300x139.jpg" alt="" title="Playing with Open Data- a simple viewer for DecoroUrbano.org" width="300" height="139" class="aligncenter size-medium wp-image-661" />
			</li>
		</ul>
		<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
		<script src="js/mg-jquery-pinit.js"></script>
		<script>
		$('ul img').pinit({btnImgUrl: 'img/ribbon-red-on-white.png'});
		</script>

If the pinned image has a title attribute it is used for a default pin description.

Requirements

  • All modern browsers
  • Internet Explorer >= 8
  • jQuery >= 1.2.3
